# Hola Soy Pamela Jesabel Carriel

Soy desarrolladora con una sólida base en programación, análisis de datos y administración de sistemas.

Me apasiona construir soluciones tecnológicas que sean eficientes, seguras y visualmente atractivas, integrando la lógica del backend con la experiencia del usuario en el frontend.
